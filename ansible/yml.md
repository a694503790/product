## yml语法

	1. 大小写敏感
	2. 使用缩进代表层级关系
	3. 缩进不允许使用tab，只能空格
	4. 缩进需注意左侧对齐

### 对象

```
msg: 您好
```
### 数组

```
- ip
- mac
- dns
```

### 复合使用
#### 对象+对象
```
msg: 您好
  ip: 1.1.1.1
  mac: "00:50:56:aa:b3:4d"
  dns: 114.114.114.114
```

#### 对象+数组
```
msg:
  - 
    name: qiumin
  - 
    name1: qiumin1
  - 
    name2: qiumin2
```

#### 数组+数组
```
- ip
  - ip1
  - ip2
- mac
  - mac1
  - mac2
- dns
  - dns1
  - dns2
```
#### 数组+对象
```
-
  ip1: 1.1.1.2
  ip2: 1.1.1.3
-
  mac1: 1.1.1.2
  mac2: 1.1.1.3
-
  dns1: 4.4.4.4
  dns2: 8.8.8.8
```
